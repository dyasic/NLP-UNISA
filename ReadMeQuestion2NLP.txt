Instructions to run Question2NLP Program:

1. Extract Question2NLP.zip ZIP file.
2. Navigate to .\Question2NLP\Question2NLP\Question2NLP\bin\Debug
3. Run Question2NLP.exe
4. Wait for the output from the Console to finish running.
5. Press Enter and you will be presented with a Menu.
6. Select 1 to implement The Most Likely Tag, the results of that method will be available in the console output, or you can find them in a text file in the path .\Question2NLP\Question2NLP\Question2NLP.
7. Select 2 to create a new Annotated file, the directory and name of the output file should be entered on the line "Please enter the file path", and then press Enter.
8. Select 3 to implement the Confusion Matrix.
